NutriFit AI is an AI-powered mobile application designed to revolutionize personal fitness and nutrition. It provides tailored meal plans and fitness routines based on user-specific data, such as age, height, weight, gender, and health goals. The app integrates real-time pose estimation to ensure proper workout form, preventing injuries and enhancing effectiveness.

Key Features:

Personalized Diet Plans: Generates customized 3-meal plans based on user health metrics and fitness goals.

Real-time Posture Feedback: Uses MediaPipe to track 33 body landmarks, providing instant feedback on exercise form and posture.

AI Chatbot: Offers motivation, tracks progress, and answers user queries.

Cheat Meal Detection: Allows users to upload images of meals, analyzes caloric intake, and adjusts future meals or workouts to maintain balance.

Adaptive Plan Updates: The system updates diet and workout plans every 15 days based on user progress.

Technologies Used:

Flutter for seamless cross-platform UI.

Python AI Models (XGBoost, MediaPipe) for nutrition and fitness recommendations.

TensorFlow Lite and MobileNet for calorie estimation via image recognition.

ChatGPT API for conversational engagement and personalized coaching.

NutriFit AI is built to provide a comprehensive, engaging, and adaptive health management solution, making it easier for users to maintain a healthy lifestyle
